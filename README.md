# Portafolio web — Romina de la Torre · Versión 2

Sitio editorial, responsive y sin dependencias externas. La identidad visual se construye a partir del borgoña, blanco cálido, negro y grises presentes en el portafolio original.

## Abrir el proyecto

Puedes abrir `index.html` directamente. Para probarlo como un sitio publicado:

```bash
python -m http.server 8080
```

Luego visita `http://localhost:8080`.

## Estructura

- `index.html`: estructura, contenido y accesibilidad.
- `styles.css`: sistema visual, diseño responsive y animaciones.
- `script.js`: menú, navegación activa, presentación interactiva y galería.
- `assets/portfolio/*.webp`: 52 láminas optimizadas para web.
- `assets/profile.webp`: retrato extraído del portafolio.
- `assets/favicon.svg`: icono del sitio.

## Mejoras incluidas

- Presentación editorial de proyectos mediante imagen fija y contenido desplazable.
- Galería de una sola lámina con flechas, teclado, gesto horizontal, zoom y selector de páginas.
- Optimización de imágenes: el sitio pasó de aproximadamente 141 MB a cerca de 8 MB en recursos visuales.
- Menú móvil completo, barra de progreso y navegación activa.
- Datos de contacto reales incluidos en el portafolio.
- Mejor accesibilidad: etiquetas, foco restaurado, teclado y reducción de movimiento.
- Diseño adaptable para escritorio, tablet y celular.
